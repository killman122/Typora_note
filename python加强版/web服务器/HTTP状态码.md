# HTTP状态码的介绍

HTTP状态码__用于表示web服务器响应状态的3位数字代码__

| 状态码 | 说明                            |
| ------ | ------------------------------- |
| 200    | 请求成功                        |
| 307    | 重定向                          |
| 400    | 错误的请求,请求地址或者参数有误 |
| 404    | 请求资源在服务器不存在          |
| 500    | 服务器内部源代码出现错误        |
| 501    | 服务器不具备完成请求的功能      |

**浏览器对303状态码的处理跟原来浏览器对HTTP1.0的302状态码的处理方法一样；浏览器对307状态码处理则跟原来HTTP1.0文档里对302的描述一样。**
